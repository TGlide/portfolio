## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

`yarn` installed on your local machine.

### Installation

#### 1. Clone the repo

```sh
git clone https://github.com/TGlide/portfolio
cd portfolio
```

#### 2. Install dependencies

```sh
yarn
```

## Usage

### Run App

```sh
yarn dev
```
